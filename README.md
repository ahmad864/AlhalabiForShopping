 Alhalabi For Shopping 

مشروع متجر إلكتروني مبني باستخدام ASP.NET Core MVC بلغة C#.

 ## فكرة المشروع
تطبيق ويب للتسوق الإلكتروني يسمح للمستخدمين بتصفح المنتجات
وعرض تفاصيلها من خلال واجهة مستخدم بسيطة.

## متطلبات التشغيل
لتشغيل المشروع يجب توفر:
- Visual Studio 2022
- .NET 8.0 SDK
- (اختياري) SQL Server أو LocalDB

##  طريقة تشغيل المشروع
1. افتحVisual Studio  
2. اختار Open a project or solution  
3. افتح الملف: AlhalbiForShopping.sln
4. اضغط Run (F5)  
5. سيتم فتح المشروع تلقائيًا على المتصفح

## قاعدة البيانات 
استخدام قاعدة بيانات:
1. افتح Package Manager Console
2. نفذ الأمر:Update-Databas
